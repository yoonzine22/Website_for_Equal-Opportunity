# 2027학년도 기초생활수급자 등 특별전형 대학 안내 웹사이트 구축 계획

## 1. 프로젝트 개요
*   **목적:** 2027학년도 기초생활수급자, 차상위계층, 한부모가족 지원대상자 특별전형 정보를 제공하는 웹사이트 개발
*   **타겟 대상:** 해당 전형에 지원하고자 하는 수험생 및 학부모
*   **데이터 출처:** 제공된 PDF 파일 (`2027학년도 기초생활수급자, 차상위계층, 한부모가족 지원대상자 특별전형.pdf`)
*   **저장 위치:** `~/Desktop/Website for University/`

## 2. 주요 기능 및 요구사항
1.  **카테고리 분류:**
    *   모집 시기별: 수시 / 정시
    *   지역별 (서울, 경기, 인천, 강원, 대전, 충남, 충북 등)
2.  **대학별 상세 정보 표시:**
    *   전형 유형 (교과, 종합 등)
    *   전형명
    *   모집 인원
    *   전형 방법 (학생부, 면접, 서류 비율 등)
    *   수능 최저학력기준 (있는 경우 표기, 비고란 활용)
3.  **검색 기능:** 대학명, 지역, 전형명 등으로 검색할 수 있는 기능

## 3. 기술 스택 (제안)
*   **Frontend:** HTML, CSS, JavaScript (Vanilla JS 또는 가벼운 라이브러리 사용)
*   **Data Structure:** 추출된 PDF 데이터를 JSON 형식으로 변환하여 JavaScript에서 활용
*   **Styling:** 심플하고 가독성 높은 CSS (CSS Grid/Flexbox 활용)

## 4. 개발 단계 (진행 순서)

### Phase 1: 데이터 추출 및 전처리 (Data Extraction)
*   [x] PDF 파일에서 텍스트 추출 (`data.txt` 생성 완료)
*   [ ] 추출된 텍스트(`data.txt`)를 파싱하여 구조화된 JSON 데이터(`data.json`)로 변환.
    *   수시/정시 구분 로직 필요 (텍스트 내 "수시모집", "정시모집" 키워드 활용)
    *   표 형태의 데이터를 객체 배열로 변환: `[{ region, university, type, name, recruitNum, method, note/수능최저 }]`

### Phase 2: 웹사이트 구조 설계 및 마크업 (HTML/CSS)
*   [ ] `index.html` 생성
    *   검색창 (Search Bar)
    *   필터링 옵션 (수시/정시 라디오 버튼 또는 드롭다운, 지역별 선택 드롭다운)
    *   데이터를 표시할 테이블 구조 설계 (`<table id="dataTable">`)
*   [ ] `style.css` 생성
    *   사용자 친화적이고 직관적인 UI 디자인 적용 (깔끔한 표 형태)

### Phase 3: 기능 구현 (JavaScript)
*   [ ] `script.js` 생성
    *   `data.json` 파일을 fetch하여 로드
    *   데이터 렌더링 함수 구현 (조건에 맞는 데이터를 HTML 테이블에 삽입)
    *   검색 기능 구현 (입력된 텍스트로 데이터 필터링)
    *   카테고리 필터링 기능 구현 (수시/정시, 지역 선택 시 데이터 재렌더링)

### Phase 4: 테스트 및 검증
*   [ ] 브라우저에서 `index.html`을 열어 데이터가 정확히 표시되는지 확인
*   [ ] 검색 및 필터 기능 정상 작동 여부 테스트
